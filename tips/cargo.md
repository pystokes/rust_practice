# Cargo

## Create new project

```bash
cargo new PROJECT_NAME --bin
```

## Build

```bash
# Build only -> @ target/debug/BINARY
cargo build
```

## Build and run

```bash
# Build and run
cargo run
```

## Check (Not compile)

```bash
# Check compilable or not
# Run compile but not create binary
cargo check
```

## Build as release version

```bash
# Release (Create optimized binary)
cargo build --release
```

## Update Cargo.lock

```bash
# Update Cargo.lock
cargo update
```
